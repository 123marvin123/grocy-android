<p>Grocy Android ist ein quelloffener Android-Client für <a href="https://grocy.info/">grocy</a> (<a href="https://github.com/grocy/grocy">Quellcode</a>). grocy ist eine Webanwendung zur Lebensmittel- und Haushaltsverwaltung <b>zum Selbsthosten</b>.</p>
<p>Grocy Android benutzt grocy's offizielle API, um Ihnen auf dem Smartphone eine native Benutzeroberfläche bereitzustellen, zusammen mit einem schnellen Barcode-Scanner und Stapelverarbeitung. Alles was Sie brauchen, um Ihre Lebensmittel effizient zu verwalten.</p>
<p><br><b>Diese App benötigt eine selbst gehostete Instanz der grocy-Webanwendung. Sie dient als Zusatz und kann daher keine Lebensmittel ohne einen Server verwalten. Sie können die App aber testen, indem Sie die Demo-Option auf dem Anmeldebildschirm wählen.</b><br></p>

<p><b>Fähigkeiten:</b></p><ul>
   <li>Bestandsübersicht</li>
   <li>Einkaufsliste mit Offline-Unterstützung</li>
   <li>Einkaufsmodus (große Klickelemente)</li>
   <li>Schnelles Barcode-Scannen</li>
   <li>Einbindung von OpenFoodFacts</li>
   <li>Bearbeitung der Stammdaten</li>
   <li>Intuitive Stapelverarbeitung</li>
   <li>Dunkler Modus</li>
   <li>Keine Werbung, Analyse oder In-App-Käufe</li>
   <li>Geringe Datenvolumen-Nutzung</li>
   <li>Kleine App-Größe (~6MB)</li>
</ul>

<p><br><b>Beteiligung:</b></p>
<p>Wenn Sie einen Fehler bemerken oder Verbesserungsvorschläge haben, zögern Sie nicht, <a href="https://github.com/patzly/grocy-android/issues">ein neues Issue</a> auf GitHub zu öffnen, uns in der App Feedback zu geben, oder eine E-Mail zu schreiben. Wie die grocy-Webanwendung, kann auch dieses Projekt übersetzt werden. Bitte besuchen Sie unsere <a href="https://github.com/patzly/grocy-android">GitHub-Seite</a> für mehr Informationen.</p>

<p><br><b>Kompatibilität:</b></p>
<p>Grocy Android benötigt mindestens grocy 2.7.0 auf Ihrem Server. Es ist auch möglich, das grocy Add-on auf einem Hass.io-Server zu verwenden. Die FAQ beschreibt, wie dies funktioniert.<br></p>
